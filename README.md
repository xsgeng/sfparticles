## 安装方法

### 克隆仓库
```bash
git clone http://172.16.95.131/gengxs/sfparticles.git
```

然后用户安装
```bash
cd sfparticles
# editable install
pip install --user -e .
```

### 直接安装
```bash
pip install --user git+http://172.16.95.131/gengxs/sfparticles.git
```
